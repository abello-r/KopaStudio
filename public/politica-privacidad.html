Política de Privacidad

En Kopa Studio tratamos datos personales para atender consultas comerciales y prestar servicios de desarrollo web, bots y automatización.

Datos que podemos tratar

  • Datos de contacto facilitados por el usuario (nombre, teléfono, email).
  • Información del proyecto compartida durante el proceso comercial.

Finalidad y legitimación

Usamos los datos para responder solicitudes, preparar propuestas y ejecutar servicios contratados. La base jurídica es el consentimiento y/o la ejecución de una relación precontractual/contractual.

Conservación

Los datos se conservarán el tiempo necesario para atender la solicitud y cumplir obligaciones legales.

Derechos

Puedes solicitar acceso, rectificación, supresión, oposición o limitación del tratamiento contactando por WhatsApp.

Terceros

No se ceden datos a terceros salvo obligación legal o servicios técnicos necesarios para la prestación.
